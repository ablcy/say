# 更新日志

## [V2.0.0] - 2026-05-04

### 新增功能
- 合并消息界面和通讯录界面，统一在消息界面展示
- 搜索框直接搜索用户账号即可添加好友
- 搜索后先显示用户头像、账号等信息，点击"添加"按钮才真正添加
- 所有添加的好友都保存在消息界面，不会因更新丢失
- 进入聊天界面时自动滚动到最底部，显示最新消息
- 导航栏底部"我"改为"个人"
- 新用户注册后自动添加 YanTalk 官方为好友并发送欢迎消息

### 问题修复
- 修复头像每次刷新后消失的问题 - 头像改为 Base64 格式存储在数据库
- 登录时返回完整用户信息（包括头像和昵称）

### 优化改进
- 优化添加好友交互流程，先展示搜索结果再添加
- 优化注册后消息加载速度

- 状态：正式发布
